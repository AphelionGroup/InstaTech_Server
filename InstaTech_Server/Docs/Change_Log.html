InstaTech Server Change Log

Version 1.4.5

  • Removed WebRTC from Electron clients.
  • .NET clients now work on Windows 7.

Version 1.4.3

  • Changed data transfer process to improve performance on slower connections.
  • Increased maximum virtual screen size.
  • Minor UI changes.

Version 1.4.2

  • Added persistent sessions for interactive console.

Version 1.4.1

  • Added console command history (up and down arrows).
  • Fixed a bug with escaping quotes in interactive PowerShell console.

Version 1.4.0

  • Added interactive scripting console.
  • Added Last Logged-On User to search matches.
  • Fixed bug with process not exiting when deploying a file.

Version 1.3.1

  • Fixed computer hub scrolling after new search.
  • Fixed a bug that would allow multiple connections from the same account to remain open.
  • Fixed horizontal scrolling on computer hub.

Version 1.3.0
